# xmcp-dev/skills

Skills for building MCP servers with [xmcp](https://xmcp.dev).

## What are Skills?

Skills are modular packages that extend Claude's capabilities with specialized knowledge and workflows for xmcp development.

## Available Skills

| Skill | Description |
|-------|-------------|
| [tool-design](./skills/tool-design/) | Create xmcp tools with schema, metadata, and handlers |
| [mcp-server-design](./skills/mcp-server-design/) | Design MCP servers with agent-friendly tools |
| [resource-design](./skills/resource-design/) | Create MCP resources to expose content for LLM consumption |
| [prompt-design](./skills/prompt-design/) | Create MCP prompts as reusable prompt templates |
| [widget-design](./skills/widget-design/) | Design UI widgets for GPT Apps and MCP Apps |

## Installation

```bash
npx add skill xmcp-dev/skills
```

Or copy the skill contents directly into your Claude project.
